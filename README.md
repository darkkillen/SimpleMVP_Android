# SimpleMVP_Android

Basic structure for MVP pattern.

:including:
  - Base structure
  - simple HTTP Manager
  - MVP template for
      - Activity
      - Fragment
      - Pager adapter
      - RecyclerView Adapter with viewholder
  - simple utils
  - simple widget and viewgroup template
  - Sample Model
  - Little Sample how MVP worked
